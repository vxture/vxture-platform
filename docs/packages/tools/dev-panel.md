# @vxture/dev-panel

> 仅用于本地开发环境，不部署至生产。

---

## 包信息

| 项     | 值                                    |
| ------ | ------------------------------------- |
| 包名   | `@vxture/dev-panel`                   |
| 路径   | `tools/dev-panel/`                    |
| @layer | `Tools`（开发工具，不属于任何业务层） |
| 端口   | 8090                                  |

## 职责

本地可视化服务控制面板，统一管理开发环境中所有服务的生命周期：

- 启动 / 停止 / 重启单个服务
- 一键全部启动 / 全部停止
- 按依赖顺序启动（BFF → Portal）
- 查看最近日志输出
- 接口级健康检查（不只检查端口，还验证关键接口返回码）
- 一键打开服务地址

## 启动

```bash
pnpm dev:panel
# → http://localhost:8090
```

## 说明

- 此工具仅为开发便利而存在，不影响任何业务逻辑
- 生产环境使用 K8s / Docker Compose 管理服务
- 如需添加新服务，在 `tools/dev-panel/src/server.mjs` 中注册
